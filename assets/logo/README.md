# Logo — Órulo

Os arquivos `orulo-azul.svg` e `orulo-branco.svg` aqui são **placeholders**
gerados em Poppins para o guia funcionar de imediato.

**Substitua pelos arquivos oficiais** do wordmark (com o pingo correto sobre o
"o") assim que possível, mantendo os mesmos nomes:

- `orulo-azul.svg` — para fundos claros
- `orulo-branco.svg` — para fundos escuros/azul

Regras de uso (ver `docs/diretrizes-de-criacao.md` §6):

- **Só** azul `#0037FF` ou branco. Nunca preto, cinza, neon ou cores especiais.
- Proporção **≈ 3.32:1**. Nunca distorcer.
- Assinatura discreta, no rodapé. Preservar respiro ao redor.
